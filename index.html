What is Wallet Refresh

Wallet Refresh is an open source protocol for connecting decentralised applications to mobile wallets with QR code scanning or deep linking. A user can interact securely with any Dapp from their mobile phone, making Wallet Refresh wallets a safer choice compared to desktop or browser extension wallets.




How does it work

Wallet Refresh connects web applications to supported mobile wallets. Wallet Refresh session is started by a scanning a QR code (desktop) or by clicking an application deep link (mobile).

Frequently Asked Questions


How do I install Wallet Refresh?


Wallet Refresh is not an app, but a protocol supported by many different decentralised applications and wallets. Install any of mobile wallets supporting Wallet Refresh protocol. Wallet Refresh wallets are available for Android and iPhone.



Is there a token?


There is no token. Wallet Refresh protocol does not run on a blockchain and there are no fees.

How can I build a wallet supporting Wallet Refresh?


There are libraries for React-Native(Javascript), Android (Java) and Swift (iOS) available. Read more about it in our documentation.



How can I add Wallet Refresh support to Dapp I developed?


You can support various wallets either usingWeb3Modal library or add a support for Wallet Refresh provider directly.



How web frontend and mobile wallets communicate?


Communication happens over a bridge server that relays messages without access to their content. The contents are encrypted using the session data shared by the QR code or deep link between the dapp and the wallet. Read more about it in ourdocumentation. Wallet Refresh Association runs a public bridge server, but you can also roll your own.
